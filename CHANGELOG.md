## [1.0.0] - 2025-01-01

### Added
- Core autonomous routing engine
- Provider connectors: AWS, Azure, CoreWeave, Lambda Labs
- CLI with full command set (init, connect, run, savings, dashboard, billing)
- MCP server for AI agent integration
- Real-time savings calculator with transparent breakdown
- Idle GPU monetization engine
- Automatic billing with USDT (TRC20) payment page
- TRON blockchain payment verification
- Real-time dashboard (port 8080)
- Shortage prediction intelligence engine
- One-click deployment script
- Full test suite
- Apache 2.0 license
- Governing documentation

### Security
- Zero credential storage — API keys never leave user infrastructure
- Zero workload data access — compute metadata only
- Minimum permission model for all provider connections
